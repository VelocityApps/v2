# Customer Onboarding Playbook

## Overview

Exceptional onboarding sets the foundation for customer success. This playbook ensures every customer gets up and running quickly, sees value immediately, and stays engaged long-term.

**Goal:** 70%+ activation rate (install 1+ automation within 24 hours), 50%+ 7-day retention

---

## Onboarding Philosophy

**Principles:**
- ⚡ **Fast time to value** (see results in first hour)
- 🎯 **Clear next steps** (no confusion)
- 🎉 **Celebrate wins** (motivate continuation)
- 📚 **Learn by doing** (not just reading)
- 🔍 **Proactive support** (help before they ask)

---

## Onboarding Stages

### Stage 1: Signup (Day 0)

**Goal:** Get customer to signup, activate account

**Signup Flow:**
1. Landing page (value prop, social proof)
2. Signup form (email only for free, credit card for paid)
3. Email verification (if required)
4. Welcome email (next steps)

**Welcome Email (Immediate):**
```
Subject: Welcome to VelocityApps! 🚀

Hi [Name],

Welcome to VelocityApps! I'm [Your Name], founder. I'm here to help you automate your Shopify store.

Getting started is simple - 3 steps:

1. Connect your Shopify store (2 minutes)
   → [Connect Now]

2. Pick an automation (we recommend starting with Review Requests)
   → [View Automations]

3. See it work (your first automation will run within 1 hour)

Need help? Just reply to this email - I personally respond to every message within 2 hours.

Let's get you set up: [Dashboard]

Best,
[Your Name]
Founder, VelocityApps

P.S. Check out our 2-minute demo video: [Link]
```

**Success Metrics:**
- Signup completion rate: >80%
- Email open rate: >50%
- Click-through rate: >30%

### Stage 2: Activation (Day 0-1)

**Goal:** Customer installs 1+ automation

**Activation Flow:**

**Step 1: Connect Shopify (2 minutes)**
- Click "Connect Shopify" button
- Shopify OAuth flow
- Store connected successfully

**On-Screen Guidance:**
```
✅ Store Connected: [Store Name]

Great! Now let's set up your first automation.

[Continue to Automation Selection]
```

**Step 2: Automation Selection (3 minutes)**
- Show recommended automation (Review Requests)
- Explain why (highest ROI, easiest to see results)
- Show preview (what it does)
- Click "Add to Store"

**On-Screen Guidance:**
```
🎯 Recommended: Review Request Automator

This automation will:
• Send review emails automatically (3-7 days after purchase)
• Increase review rate from 2% to 8% (industry average)
• Save you 2 hours/week

[Add to Store] [See Other Automations]
```

**Step 3: Configuration (5 minutes)**
- Simple configuration (defaults work for most)
- Optional: Customize (email template, timing)
- Click "Activate"

**On-Screen Guidance:**
```
⚙️ Configure Review Request Automator

Default settings work for most stores. You can customize later.

Send delay: 3 days after delivery ✓
Email template: Default ✓
Include discount: 10% off ✓

[Activate Now] [Customize Settings]
```

**Step 4: Activation Confirmation (Immediate)**
- Show success message
- Explain what happens next
- Set expectations (first email sent when?)

**On-Screen Guidance:**
```
🎉 Automation Activated!

Your Review Request automation is now running.

What happens next:
• Your next order will trigger the automation
• Review emails will send 3 days after delivery
• You'll see results in your dashboard within a week

[View Dashboard] [Add Another Automation]
```

**Success Metrics:**
- Connection rate: >90% (complete Shopify connection)
- Automation selection: >80% (select 1+ automation)
- Activation rate: >70% (activate 1+ automation within 24 hours)

## Onboarding Optimization

### A/B Tests

**Test 1: Welcome Email Timing**
- Variant A: Immediate (on signup)
- Variant B: Delayed (1 hour after signup)
- Metric: Activation rate

**Test 2: Recommended Automation**
- Variant A: Review Requests (highest ROI)
- Variant B: Low Stock Alerts (easiest to see results)
- Metric: Activation rate

**Test 3: Configuration Complexity**
- Variant A: Simple (defaults only, activate immediately)
- Variant B: Guided (walk through configuration)
- Metric: Activation rate, time to activate

**Test 4: Onboarding Emails**
- Variant A: 3 emails (Day 0, 1, 7)
- Variant B: 5 emails (Day 0, 1, 3, 7, 14)
- Metric: Engagement, retention

---

### Drop-Off Analysis

**Track Drop-Off Points:**
1. Signup → Email verification (if applicable)
2. Email verification → Dashboard visit
3. Dashboard → Shopify connection
4. Shopify connection → Automation selection
5. Automation selection → Configuration
6. Configuration → Activation

**Identify Patterns:**
- Where do customers drop off most?
- What causes drop-off?
- How to reduce drop-off?

**Optimization:**
- Simplify drop-off steps
- Add guidance at drop-off points
- Send re-engagement emails (if inactive)
