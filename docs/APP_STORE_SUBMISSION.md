### Review timeline
Shopify App Store review typically takes **5–10 business days** for a first submission. Common rejection reasons:
- Screenshots don't match app functionality → use real screenshots
- Privacy policy doesn't mention Shopify data → yours does, you're fine
- App errors during review → make sure the install flow works on a fresh dev store
- Missing GDPR webhooks → all implemented and tested
